## Desenvolvimento Local

1.  Clone o repositório.
2.  Instale as dependências: `npm install`
3.  Rode o servidor de desenvolvimento: `npm start` (ou `ng serve`)
4.  Abra `http://localhost:4200/` no seu navegador.
